# <center>实用计算器</center>

<center><font face="courier New" color="#2e7d32" size="6">Useful Tools</font></center>

> [!TIP] 💡 须知
> 本页面整合了服务器常用的实用工具，帮助你在游玩过程中快速估算费用、模拟抽取体验。所有工具均在浏览器本地运行，数据不会上传。

## 一、领地计算器

不同玩家称号的圈地**最大范围**与**价格**各不相同。选择你的称号等级并输入尺寸，即可快速估算圈地所需费用。

<Callout type="info" title="💰 如何使用">
1. 点击下方对应的**称号按钮**（PLAYER / VIP / MVP / PRO / HERO）
2. 输入你想要圈地的**长度**和**宽度**（单位：格）
3. 计算器将自动显示**占地格数**与**预估费用**
</Callout>

<ResidenceCalculator />

<Callout type="info" title="圈地小贴士">
- 圈地的具体操作方法与指令，详见 [领地与地标](/index/新手指南/领地与地标.md) 页面。
- Y 轴高度计入领地但不额外计费，通常只需用平面对角线两点圈地即可。
- 称号越高，单价越低、可圈范围越大，详见 [头衔称号](/index/基础信息/头衔称号.md)。
</Callout>


## 二、转账汇率计算器

服务器内玩家之间转账会收取 **8%** 的手续费（由转账方承担），但**低于 50 金币**的转账免税。输入你想转账的金额，即可快速算出你实际需要支付多少。

<Callout type="info" title="💸 如何使用">
1. 在下方输入框中填入**转账金额**
2. 计算器会自动显示额外的**手续费**与你**实际需要支付的总金额**
3. 低于 50 金币时无手续费，无额外支出
</Callout>

<TransferCalculator />

## 三、拍卖行汇率计算器

在拍卖行售出物品时，系统会按照你的称号等级收取不同比例的手续费。称号越高，税率越低。输入你的上架售价，即可算出实际到手金额。

<Callout type="info" title="🏷️ 如何使用">
1. 点击对应的**称号按钮**选择你的称号等级
2. 输入物品的**上架售价**
3. 计算器会自动显示手续费与**实际到手金额**
</Callout>

<AuctionCalculator />

## 四、附魔书模拟抽奖

想知道附魔师 NPC 抽到好书的概率有多大？在消耗真正的附魔之眼之前，先来这里模拟体验一下吧！

<Callout type="info" title="🎲 如何使用">
1. 选择想要抽取的**品质标签**（普通 / 优秀 / 史诗 / 传说 / 神话 / 全品质）
2. 点击**抽取按钮**开始模拟
3. 观察抽到的附魔书名称、等级与保底进度
</Callout>

<EnchantLottery />

<Callout type="info" title="附魔书小贴士">
- 完整的附魔书图鉴与属性查询，前往 [附魔书查询引擎](/index/拓展玩法/附魔书查询引擎.md)。
- 附魔书的详细玩法与获取方式，参见 [拓展附魔书系统](/index/拓展玩法/拓展附魔书系统.md)。
- 本模拟器仅供娱乐参考，实际抽取结果以游戏内为准。
</Callout>

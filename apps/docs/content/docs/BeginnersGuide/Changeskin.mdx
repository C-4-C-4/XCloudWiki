# <center>更改皮肤</center>

<center><font face="courier New" color="#1f009bff" size="6">Change Skin</font></center>


## 一、如何更换皮肤

进入游戏后，您可以按 **T** 键打开聊天框，输入以下指令来管理您的皮肤：

### 基础信息
- **核心指令**：`/skin` (基础操作)
- **皮肤菜单**：`/skins` (打开图形化选择界面)
- **刷新皮肤**：`/skin update` (当皮肤显示异常时使用)

<Callout type="info" title="💡 换肤建议">
- **正版玩家**：通常无需输入指令即可自动同步。如果皮肤未及时显示，请尝试**重启客户端**或使用 `/skin update` 刷新。
- **离线玩家**：支持通过 `/skin [URL]` 使用网络皮肤直链进行更换，或使用 `/skins` 菜单快速挑选。
</Callout>

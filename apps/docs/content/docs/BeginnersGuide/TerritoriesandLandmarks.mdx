# <center>领地与地标</center>

<center><font face="courier New" color="#2e7d32" size="6">Territory & Landmarks</font></center>

> [!TIP] 💡 须知
> 领地用于保护你的建筑与资源；地标则用于向全服玩家展示并传送到你的店铺、基地或公益设施。二者配合使用，是生存服建家与经营的核心功能。


## 一、领地系统

### 1、创建领地

<Callout type="info" title="圈地工具">
请**手持木锄**，用**左键**点击一个方块作为起点，再用**右键**点击另一个方块作为对角点。两点所围成的平面区域即为预选领地范围。
</Callout>

- 目前领地**高度无上限**，选定平面后 Y 轴整列高度均计入领地，因此通常只需用平面对角线两点圈地即可。
- 选区完成后，屏幕会显示**圈地范围**与**所需金币**。
- 确认无误后，输入 `/res create [自定义名字]` (名称仅英文与数字，不可包含特殊符号)即可创建属于你的领地。

![](/wiki-img/BeginnersGuide/TerritoriesandLandmarks/2.png)

---

### 2、领地指令一览

<p className="cmd-section-desc">以下指令按功能分类展示；标注 <span className="cmd-tag">需站在领地内</span> 的须在对应领地中执行，<span className="cmd-tag global">全局</span> 可在任意位置使用。<br/>点击下方指令即可复制！</p>

#### 📦 创建与设置
<br/>
<div className="command-grid">
  <div className="cmd-card">
    <div className="cmd-name">/res create [名字]</div>
    <div className="cmd-desc">在木锄选定的区域内创建领地，名称仅支持英文与数字。</div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res remove</div>
    <div className="cmd-desc">删除当前站立所在的领地。<span className="cmd-tag">需站在领地内</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res set</div>
    <div className="cmd-desc">打开领地设置面板，可配置回血、禁足、禁止传送等选项。<span className="cmd-tag">需站在领地内</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res expand [格数]</div>
    <div className="cmd-desc">面向需要扩大的方向延伸领地边界。<span className="cmd-tag">需站在领地内</span></div>
  </div>
</div>

---

#### 👥 权限管理
<br/>
<div className="command-grid">
  <div className="cmd-card">
    <div className="cmd-name">/res padd [玩家ID]</div>
    <div className="cmd-desc">授予指定玩家进入与使用领地的权限。<span className="cmd-tag">需站在领地内</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res pdel [玩家ID]</div>
    <div className="cmd-desc">移除指定玩家的领地权限。<span className="cmd-tag">需站在领地内</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res pset [玩家ID]</div>
    <div className="cmd-desc">针对特定玩家进行领地权限设置</div>
  </div>
</div>

---

#### 🧭 传送与查询
<br/>
<div className="command-grid">
  <div className="cmd-card">
    <div className="cmd-name">/res tpset</div>
    <div className="cmd-desc">将当前位置设为领地传送点，配合 <code>/res tp</code> 回家，不占用 home 位。<span className="cmd-tag">需站在领地内</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res tp [领地名字]</div>
    <div className="cmd-desc">传送至指定领地。<span className="cmd-tag warn">无法跨区传送</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res limits</div>
    <div className="cmd-desc">查询当前称号下可圈地的最大范围与相关上限。<span className="cmd-tag global">全局</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/res list</div>
    <div className="cmd-desc">列出你拥有权限的所有领地，包含他人授权给你的领地。<span className="cmd-tag global">全局</span></div>
  </div>
</div>

### 4、特别提醒

<Callout type="error" title="请务必注意">

- 领地传送` /res tp [领地名字] `**不支持跨世界/跨分区**，请在目标生存分区内完成圈地或传送。
- **严禁**恶意破坏他人领地及周边地形；违规行为将依据 [服务器规定](/index/基础信息/服务器规定(必看).md) 予以处理。

</Callout>


## 二、地标系统

### 1、什么是地标？

<Callout type="info" title="玩家地标 · Player Warp">
专为**个人商店**、**特色建筑**与**公益设施**设计的全服公共传送点。在店铺或机器旁购买并挂载后，其他玩家可通过地标列表或 `/pw` 指令一键到达你的位置——相当于一个可交互的「活广告」。
</Callout>

![](/wiki-img/BeginnersGuide/TerritoriesandLandmarks/1.png)

### 4、相关指令

<div className="command-grid">
  <div className="cmd-card">
    <div className="cmd-name">/pw [地标名称]</div>
    <div className="cmd-desc">快速传送到指定的公共玩家地标（Player Warp）。</div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/pw desc set [地标名称] [描述]</div>
    <div className="cmd-desc">为指定地标设置说明文字，供地标列表中展示。</div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/pw set [地标名称]</div>
    <div className="cmd-desc">将地标传送点设置到当前站立位置。<span className="cmd-tag">需在目标位置执行</span></div>
  </div>
  <div className="cmd-card">
    <div className="cmd-name">/pw remove [地标名称]</div>
    <div className="cmd-desc">移除已挂载的指定地标。</div>
  </div>
</div>

<Callout type="warn" title="与「家」的区别">
`/sethome` 与 `/home` 设置的是**私人传送点**，仅自己使用；`/pw` 传送的是**全服公开的玩家地标**，供所有玩家浏览与访问。领地传送 `/res tp` 则绑定在圈地区域内，三者用途不同，请按需选择。
</Callout>

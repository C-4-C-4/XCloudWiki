# 四、种植方式
<div className="farming-guide">
<div className="guide-steps">

<div className="step-content">
<h4>🌤️ 辨识当前季节</h4>

<div className="step-num">2</div>
<div className="step-content">
<h4>🚧 圈地与防护措施</h4>
<p>开垦原版耕地作为农田。为了防止乌鸦来偷吃作物，强烈建议在农田的四角放置 <strong>稻草人</strong> 进行防范。注意：稻草人只保护周围7格平齐的农作物！</p>
</div>
</div>
<div className="step-item">
<div className="step-num">3</div>
<div className="step-content">
<h4>🧪 提前施加肥料</h4>
<p>如果你购买了肥料或增产剂，<strong>必须在播种前</strong>拿着化肥对准耕地右键使用。每 1 个耕地块需要消耗 1 包化肥。</p>
</div>
</div>
<div className="step-item">
<div className="step-num">4</div>
<div className="step-content">
<h4>💦 浇水与自动化设施</h4>
<p>每块耕地可存水5格，拿起水壶对着水源右键装水（最高容量为 4 格），然后对准耕地长按右键即可浇水。<br /><strong>进阶技巧</strong>：如果放置了洒水器，你可以用水壶右击洒水器为其充能。洒水器等级越高，全自动浇水的覆盖面积就越大！</p>
</div>
</div>
</div>

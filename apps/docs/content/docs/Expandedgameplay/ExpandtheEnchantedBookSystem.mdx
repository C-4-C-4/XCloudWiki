## 一、附魔书获取
### 1、附魔之眼兑换

<Callout type="info" title="💡 核心兑换说明">
玩家可通过 **每日任务**、**玩家市场**、**提升职业等级** 等途径获取 **<font color="#EF4444">附魔之眼</font>**。  
收集足够的附魔之眼后，前往主城的 **NPC 附魔师** 处，即可抽取兑换对应品质的附魔书（附魔属性与等级均为随机）。

**🎁 保底机制**：单品质连续抽取 **10 次** 后触发保底——普通必出 **六级**，优秀/史诗必出 **五级**，传说/神话必出 **四级**。另可 **全品质随机抽取**，连续 **20 次** 后保底必出 **六级** 附魔书。

**💰 抽取消耗**：普通 **8000 经验**；优秀 **1** 个附魔之眼；史诗 **4** 个；传说 **10** 个；神话 **15** 个。
</Callout>

# 三、附魔注意事项 

<Callout type="warn" title="⚠️ 附魔核心规则">
- **槽位机制**：每件装备初始仅有 **3 个**附魔槽。如需新增附魔，必须使用 **附魔凿** 工具开启新槽位（最大上限 8 个）。
- **品质限制**：<font color="#FBBF24">传说</font> 和 <font color="#F87171">神话</font> 品质的附魔在同一装备上**各仅限一种**！
- **附魔冲突**：部分附魔之间存在互斥（例如经验修补与无限，以及部分特殊拓展附魔），冲突会导致无法附魔，请提前在图鉴中确认。
</Callout>

# 五、附魔突破

<Callout type="info" title="🚀 突破系统简介">
**附魔突破** 能够将拓展附魔书（不包括原版附魔书）的等级上限**强行提高一级**，打破装备属性的极限！
</Callout>

<img src="/wiki-img/Expandedgameplay/ExpandtheEnchantedBookSystem/image-fm5.png" style={{ borderRadius: '8px', display: 'block', margin: '1rem auto', boxShadow: '0 4px 12px rgba(0,0,0,0.1)' }} />

### 1、突破核心规则

<div className="breakthrough-rules">
  <ul>
    <li><strong>前置条件：</strong> 只有达到<strong>极限等级</strong>的附魔才能进行突破。</li>
    <li><strong>成功概率：</strong> 每放入一本相同的满级附魔书增加 <strong>34%</strong> 概率，放入三本即可达 <strong>100%</strong> 成功率。</li>
    <li><strong>失败惩罚：</strong> 突破失败会导致<strong>所有材料（包括附魔书）消失</strong>。</li>
    <li><strong>必须材料：</strong> <font color="#EF4444"><strong>附魔进阶卷轴</strong></font> 是突破的硬性消耗品。</li>
    <li><strong>保底材料：</strong> <font color="#4ADE80"><strong>附魔突破水晶</strong></font>（非必须），若突破失败可<strong>保留原附魔书</strong>不被销毁。</li>
  </ul>
</div>

### 2、突破界面指南

<img src="/wiki-img/Expandedgameplay/ExpandtheEnchantedBookSystem/agaalgvgzyzrlj24cazg8r9i1hvwss29.png" style={{ borderRadius: '8px', width: '70%', display: 'block', margin: '1.5rem auto', boxShadow: '0 4px 12px rgba(0,0,0,0.15)' }} />

> [!IMPORTANT] 💡 界面摆放说明
> - **顶部槽位**：放置需要突破的**最高等级附魔书**。
> - **左下槽位**：放置 **<font color="#4ADE80">附魔突破水晶</font>**（非必须，用于防爆防碎）。
> - **右下槽位**：放置 **<font color="#EF4444">附魔进阶卷轴</font>**（必须）。
> - **底部按钮**：点击确认按钮前，可以查看当前的成功率以及所需的突破费用（固定为 **<font color="#F59E0B">15000 金币</font>**）。
